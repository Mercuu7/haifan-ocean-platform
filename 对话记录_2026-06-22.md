## 二、关键技术细节

### 网络问题
- GitHub 在本机无法直连（GFW 阻断 443/22 端口）
- 但 `gh` CLI 的 Go HTTP 栈可以通过 HTTPS 访问 GitHub API
- 因此 `git push` 不行，但可以用 `gh api` 上传文件

### 推送工具
创建了 `push_via_api.py` 脚本，用法：
```bash
python push_via_api.py index.html dashboard.html
```

## 六、Streamlit 视觉重设计（第二轮）

### 背景
用户反馈 Streamlit 页面"太丑"，AI 紫蓝色味严重，要求参考优秀仿真页面进行视觉升级。

### 第一次尝试：深海工业风（批量正则替换）
- 方法：Python 正则批量替换颜色值
- 结果：**失败** — 色调浑浊不协调，用户反馈"更丑了"

### 第二次尝试：Forge 工业锻造风（从零手写 CSS）
- 调研参考：OpenBridge 5.0（海事行业标准）、Forge Design System（Uiverse）、Tactical Telemetry（军事HUD）、纯黑铬金
- 用户选择：**Forge 工业锻造风**
- 方法：完整重写 CSS 块（非正则替换）
- 设计原则：
  - 石墨黑板 `#0c0c0c` → `#161616`
  - **单一**烬橙强调色 `#e07830`（仅用于 KPI 数据）
  - 实色卡片 + 1px 发丝线边框（无玻璃态/发光）
  - 工业小圆角 `4-8px`
  - 移除所有动画（纯静态工业感）
  - 铜绿 `#6b9e7a` 状态色

### 备选方案（可随时切换）
| 方案 | 底色 | 强调色 |
|------|------|--------|
| OpenBridge 夜航 | 深海夜蓝 `#0a1628` | 琥珀 `#e8a840` |
| 战术遥测终端 | CRT 黑 `#0a0a0a` | 磷光绿 `#4af626` |
| 纯黑铬金 | 纯黑 `#080808` | 铬金 `#c8a455` |

### 最终状态
- **静态 HTML 页面**：保持原海洋蓝设计
- **Streamlit 仿真平台**：Forge 工业锻造风（石墨黑 + 烬橙）

## 七、网络推送方案总结

由于 GFW 阻断 git push（443/22 端口均超时），所有代码通过 `gh api` 上传：

```bash
# 更新已有文件
gh api repos/Mercuu7/haifan-ocean-platform/contents/{path} \
  --method PUT \
  --input payload.json

# 创建新文件（不含 sha 字段）
gh api repos/Mercuu7/haifan-ocean-platform/contents/{path} \
  --method PUT \
  --input payload.json
```

`push_via_api.py` 脚本封装了此流程。`gh` CLI Go HTTP 栈可穿透 GFW，git/libcurl 不行。
